## Week 1: Getting Familiar with AWS and Basic Cloud Services

**Time:** 02/03/2026 - 06/03/2026

### Week 1 Objectives:

* Get familiar with the internship orientation and the requirements for the workshop/report.
* Understand the overview of AWS Cloud and the role of cloud computing.
* Understand foundational concepts such as Region, Availability Zone, AWS Management Console, and AWS Free Tier.
* Create an AWS account for learning and practice.
* Begin identifying the main groups of AWS services such as Compute, Storage, Networking, Database, Security, Monitoring, and Analytics.

### Week 1 Achievements:

* Understood that AWS is a cloud computing platform that provides many service groups for storage, computing, networking, security, monitoring, and data analytics.
* Understood basic concepts such as Region, Availability Zone, resource, service, and billing.
* Created and configured an AWS account for the internship.
* Became familiar with AWS Management Console and learned how to search, access, and use AWS services through the web interface.
* Understood the importance of cost monitoring when using AWS, especially when using services such as S3, Glue, Athena, and QuickSight.
* Identified the list of services to continue studying for the workshop.
