## Self-Assessment

During my internship at **Amazon Web Services Vietnam Co., Ltd**, I had the opportunity to learn, practice, and apply the knowledge I had gained at university in a real working environment.

I participated in the project **Building a Data Lake System and E-commerce Data Analytics Dashboard on AWS**. Through this project, I improved my skills in using AWS services such as Amazon S3, AWS Glue, Amazon Athena, Amazon QuickSight, and EventBridge Scheduler.
