Truy cập **AWS Management Console**, tìm kiếm và chọn dịch vụ **AWS Glue**. Trong menu bên trái, chọn **Data Catalog** → **Databases**.

Sau đó nhấn **Add database**.

![](/images/5-Workshop/5.8.1-Tao-Glue-Database-ecommerce-curated/image-001.png)

Tại màn hình tạo database, cấu hình như sau:

- Database type: Chọn **Glue Database**

- Name: **ecommerce_curated**

- Description: **Curated zone metadata database for cleaned ecommerce Parquet data stored in Amazon S3**.

Sau đó nhấn **Create database**.

![](/images/5-Workshop/5.8.1-Tao-Glue-Database-ecommerce-curated/image-002.png)

Glue Database **ecommerce_curated** đã được tạo thành công.

![](/images/5-Workshop/5.8.1-Tao-Glue-Database-ecommerce-curated/image-003.png)
