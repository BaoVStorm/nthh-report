# 5.17. Publish Dashboards

After completing the design and reviewing the visuals in the Analyses, it is necessary to publish to create an official dashboard.

In the QuickSight Analysis interface, click the **Publish** button.

![](/images/5-Workshop/5.17-Publish-Dashboards/image-001.png)

Select **New dashboard**, enter the dashboard name, and click **Publish dashboard**.

![](/images/5-Workshop/5.17-Publish-Dashboards/image-002.png)

Once published, the dashboard can be shared with other users if the QuickSight account has been granted appropriate permissions.
