# 5.17. Publish Dashboards

Sau khi hoàn tất thiết kế và kiểm tra các visual trong Analyses, cần publish để tạo dashboard chính thức.

Trong giao diện QuickSight Analysis, nhấn nút Publish

![](/images/5-Workshop/5.17-Publish-Dashboards/image-001.png)

Chọn New dashboard, điền tên dashboard và nhấn Publish dashboard.

![](/images/5-Workshop/5.17-Publish-Dashboards/image-002.png)

Dashboard sau khi publish có thể được chia sẻ cho người dùng khác nếu tài khoản QuickSight được cấp quyền phù hợp.
